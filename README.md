\# Chumcred Limited – Cost Management Assessment Tool



A Streamlit-based financial intelligence application that analyzes cost efficiency,

budget performance, and financial discipline using professional cost-management metrics.



\## Features

\- Upload or input financial figures

\- Computes:

&nbsp; - Cost Variance

&nbsp; - CPI

&nbsp; - OER

&nbsp; - ROI

&nbsp; - TCO

&nbsp; - Payback Period

&nbsp; - Cost to Serve

&nbsp; - Budget Utilization Rate

&nbsp; - Efficiency Ratio

&nbsp; - Gross Profit Margin

\- Interpretation + implications for decision-making

\- Downloadable Excel report

\- Lightweight, fast, and user-friendly



Built by \*\*Chumcred Limited\*\*.
